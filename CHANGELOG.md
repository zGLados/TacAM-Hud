# Changelog

All notable changes to TacAM HUD will be documented in this file.

## [1.0.0] - 2026-04-16

### Initial Release

#### Features
- ✨ Professional CS2 HUD interface
- 🎮 Full CS2 game state integration
- 📡 Real-time Socket.io communication
- 🗺️ Integrated radar with 9 map support (Dust2, Mirage, Inferno, Nuke, Overpass, Vertigo, Ancient, Anubis, Train)
- 🎯 Customizable sponsor logo rotation system
- ⚙️ OpenHud admin panel configuration
- ⌨️ Keyboard shortcuts (Alt+C, Alt+V, Alt+B)
- 🎨 TacAM branding and color scheme
- 📦 Plug-and-play ready (no build process required)

#### Assets Included
- 15 Montserrat font variants + 2 custom fonts
- Team logos and player icons (CT/T)
- 9 radar map images
- Bomb and flash assist icons
- 3 sponsor logo slots

#### Documentation
- Comprehensive README with installation guide
- Quick start guide (QUICKSTART.md)
- Sponsor customization instructions
- Troubleshooting section

#### Configuration
- `hud.json` - HUD metadata and settings
- `panel.json` - Admin panel configuration
- `keybinds.json` - Keyboard shortcuts

#### Technical Details
- React 18.2.0 based (pre-compiled)
- Socket.io for real-time updates
- Optimized and formatted CSS (2374 lines)
- Minified JavaScript bundle (645KB)
- Port 1349 (OpenHud default)
